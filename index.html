Цей гайд допоможе тобі розпочати програмування. Ми розглянемо основні принципи та синтаксис популярних мов програмування.

Базовий синтаксис

Ось приклад функції на JavaScript:


function привітання(ім'я) {
  console.log("Привіт, " + ім'я + "!");
}
привітання("Світ");

А ось аналогічна функція на Python:


def привітання(ім'я):
    print("Привіт,", ім'я)

привітання("Світ")

Приклади

Приклад програми на Python, яка перевіряє, чи є число парним:


число = int(input("Введи число: "))

if число % 2 == 0:
    print("Це парне число.")
else:
    print("Це непарне число.")
